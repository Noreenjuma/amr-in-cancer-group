# Tools required for the NGS pipeline:

# 1. FastQC: Used for quality control of raw sequencing reads
fastqc

# 2. Fastp: Used for trimming low-quality bases and adapter sequences
fastp

# 3. BWA (Burrows-Wheeler Aligner): Used to map sequencing reads to the reference genome
bwa

# 4. Samtools: Used for handling SAM/BAM files (conversion, sorting, indexing)
samtools

# 5. BCFtools: Used for variant calling and filtering of SNPs/indelsf
bcftools
